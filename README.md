# 电商平台前端项目

项目技术栈:<br/>
    <br/>1.原生Javascript<br/>
    <br/>2.原生css<br/>
    <br/>3.框架JQuery<br/>
    <br/>4.构建工具 webpack<br/>
    <br/>5.版本控制工具 git<br/>
本电商项目具体功能及模块:<br/>
    <br/>1.商品模块->首页 商品列表 商品详情<br/>
    <br/>2.购物车模块->购物车数量 添加删除商品 购物车提交<br/>
    <br/>3.订单模块->订单确认(地址管理) 订单提交 订单列表 订单详情<br/>
    <br/>4.支付模块->支付<br/>
    <br/>5.用户模块->登录 注册 个人信息 找回密码 修改密码 修改个人信息<br/>
